# Freshers Quiz

A simple web-based quiz made for college freshers events.

---

## Features

- Name & serial number based entry  
- Prevents duplicate submissions on the same device  
- MCQ-based quiz with progress bar  
- Instant score and percentage  
- Auto submission to Google Forms  
- Local storage backup if submission fails  

---

## Tech Stack

- HTML  
- CSS  
- Vanilla JavaScript  
- Google Forms  
- Browser LocalStorage  

---

## How it works

1. Enter name and serial number  
2. Start the quiz  
3. Answer all questions  
4. Get result instantly  

Results are submitted automatically.
